Desenvolvimento da Blue Pen Corporation
Como parte de um projeto acadêmico na UniALFA, criamos um site de vendas para a Blue Pen Corporation, especializada em canecas e moletons personalizados. Utilizei HTML e CSS para construir uma plataforma web que exibia os produtos, permitindo aos usuários personalizar e adquirir os produtos. O projeto representou uma oportunidade de aplicar conceitos aprendidos, criando uma experiência de compra online atrativa e funcional.

Desenvolvimento do Jogo "StrongMind"
No âmbito acadêmico da UniALFA, participamos do desenvolvimento do jogo StrongMind, um projeto voltado para a conscientização sobre saúde mental. Este jogo, criado no Construct 3, abordou questões fundamentais sobre bem-estar emocional e estratégias para lidar com o estresse do dia a dia. As animações utilizadas no jogo foram elaboradas no Canva, fornecendo elementos visuais que complementaram a narrativa interativa do jogo. A iniciativa representou uma oportunidade única para explorar o uso de diferentes ferramentas e conceitos, visando conscientizar e educar sobre saúde mental.

Desenvolvimento do Jogo "Super Ash"
Participamos ativamente, como equipe na criação do jogo Super Ash, uma iniciativa educacional voltada para o público infantil, com o tema Pokemon e foco na alfabetização. Desenvolvido no Construct 3, o jogo visava tornar o aprendizado mais envolvente e lúdico para as crianças, incentivando a alfabetização de forma divertida e interativa. Essa experiência permitiu explorar e compreender como os elementos dos jogos podem ser integrados à educação de maneira eficaz, proporcionando uma abordagem mais dinâmica no ensino para um público jovem e engajado.

Desenvolvimento do Site de Vendas da Mercedes
Participamos no desenvolvimento do site de vendas da Mercedes, utilizando HTML e CSS para criar uma plataforma responsiva e amigável para diferentes dispositivos. A proposta visava oferecer uma experiência de compra online de carros da Mercedes, integrando design responsivo para se adaptar a diversas telas, desde desktops até dispositivos móveis. Essa iniciativa permitiu explorar os fundamentos do desenvolvimento web e a importância do design responsivo na criação de interfaces acessíveis e funcionais para os usuários.

Desenvolvimento do Site de Conveniência "Carreiro"
Participamos no desenvolvimento do site de conveniência Carreiro, criado utilizando HTML e CSS. O projeto tinha como objetivo oferecer uma experiência de conveniência online para os usuários, apresentando uma interface amigável e funcional. Exploramos conceitos fundamentais de design web e estruturação de páginas, garantindo a navegabilidade e usabilidade para os visitantes. Essa oportunidade permitiu compreender como a combinação de HTML e CSS pode resultar em um site atrativo e acessível, oferecendo uma experiência prática e satisfatória aos usuários.

Desenvolvimento do Site Atacadão do Tião
Desenvolvimento do site do Atacadão do Tião, um projeto de comércio de gado e produtos agropecuários. Utilizamos HTML e CSS para criar uma plataforma online, facilitando a venda e aquisição de gado e produtos relacionados ao agronegócio. A iniciativa visava proporcionar uma interface intuitiva e informativa para os clientes interessados. A experiência me permitiu compreender como a combinação de HTML e CSS pode viabilizar um ambiente virtual eficiente e atraente para transações no setor agropecuário.
